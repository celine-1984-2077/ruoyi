# 文档站首页

这是一份托管在仓库里的 `doc.iocoder.cn` 文档镜像站，适合直接在线查阅，也适合之后继续增量同步。

## 使用方式

- 左侧导航：按模块浏览
- 顶部搜索：全文检索关键词，例如 `审计`、`BPMN`、`流程设计器`
- 页面底部原始链接：可回溯到源站地址

## 说明

- 文档源码保存在仓库内，便于版本管理和后续补充
- 页面图片已下载到本地，因此站点可以离线构建
- 如果源站文档更新，可重新运行 `scripts/fetch_iocoder_docs.py`
